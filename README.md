## Running Locally

## Quick Start
```bash
git clone https://github.com/aryanbrite/calculator-streamlit
cd calculator streamlit
pip install uv
uv sync
uv pip install -r requirements.txt
```

## Run App
```
uvx streamlit run main.py
```
